# Cinematic Portfolio | Aryan Moudgill

A high-performance, immersive freelance portfolio designed with a focus on **Liquid Glassmorphism**, **Cinematic Transitions**, and **Global Audio Strategy**. 

Built for the modern digital landscape where aesthetics meet performance.

## ✨ Features

- **Liquid Glass UI**: Premium aesthetic with deep blurs, vibrant amber accents, and fluid interactions.
- **Project Catalogue**: A structured, hierarchical showcase of digital works—from custom hosted websites to strategic social content.
- **Global Music System**: Per-page looping audio with a premium glassmorphic toggle, respecting browser autoplay policies.
- **Cinematic expertise Section**: Interactive "What am I good at?" intro featuring high-end CSS-only animations and video backgrounds.
- **Fully Responsive**: Optimized for every screen size, from mobile devices to ultra-wide displays.
- **Modern Tech Stack**: Powered by React, Vite, Framer Motion, and Tailwind CSS.

## 🛠️ Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
